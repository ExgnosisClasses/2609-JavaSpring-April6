# 2609 - Java Upskilling 
### Class Repository
### April 6. 2026

This is the class repository for the BofA / TekSystems Java Upskilling boot camp.

The materials will be freely available until June 15, 2026. You are free to download or clone the repository anytime before then.

This repository contains all the slides and labs that were distributed to you during boot camp. 

However, this repo also contains additional references, materials and supplementary information that you will hopefully find useful.

Any additional references or materials that I have introduced during the boot camp will be added to this repository.

There is also a separate lab repository that contains only the lab files from this repository. HAving a separate lab repository makes it easier for you to clone the lab file into your VM without cloning all the slides and other stuff as well.
